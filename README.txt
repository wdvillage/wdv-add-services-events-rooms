== Description ==

The plugin "WDV - Add Services, Events, Rooms" added custom post types "Services", "Abouts", "Rooms", "Testimonials", "Advertisements" to WP theme "Aniro Hotel Light".

== Installation ==

1. Upload the plugin files to the /wp-content/plugins/wdv-add-services-events-rooms directory, or install the plugin through the WordPress plugins screen directly.
2. In your WordPress administration, go to the Plugins page.
3. Activate the 'WDV - Add Services, Events, Rooms' plugin through the 'Plugins' screen in WordPress.
4. Now you can find custom post types 'Services', 'Abouts', "Rooms", "Testimonials", "Advertisements" in admin panel and you can create new services and events.
